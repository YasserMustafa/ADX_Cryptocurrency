ADX_Cryptocurrency
==================

ADX Technical Indicator for hourly OHLC of Cryptocurrency



How to Use
----------
  - Open Termnial
  - Go to directory containing adx.py
  - run 'python3 -i adx.py'
  - type command in >>> runThis()
